Teleport Your NFT's into the Metaverse!

NFT Teleporter, is an a no code service which alows users to easily create NFT's and NFT collections which they can then teleport into the Metaverse. Currently a proof of concept, our teleporter leverages IPFS's NFT Storage solutions, along with modern day 3D model file types, to allow NFT's to be viewable and interactable in VR (and soon AR). Our service opens the doorway for unique interactive experiences to be attached to NFT's (through IPFS), which can increase the value and utility of an NFT.

How it Works

No Code Custom VR Experiences

Our service allows NFT creators to easily upload relevant meta data which you would usually associate with an NFT, along with digital 3D models and other exciting data that we turn into an IPFS hosted interactable experience automatically. These experiences are then made exclusively available to white listed wallets.

Build Your NFT With a Simple UI

We empower artists and designers to build and upload their very own NFTs and NFT collections to IPFS, using our simple UI. We athenticate an artist via their Metamask wallet, and then save all relevant NFT data to IPFS. This is then stored as an object on our express server, which can be retrieved by the user either through our website UI or our REST API at any time.

Retrieve Data to Build Bespoke Experiences Later

We understand that many creators are in a rush to launch, but don't want to offer subpar experiences at the same time. The 3D model data, along with all other NFT data that we save to IPFS can be retrieved at a later point to build bespoke experiences off platform whenever it suits the creator. Up front we can provide a small set of experiences using custom data, and then the NFT creator can expand on this when they are able using our REST API.
